🚀 HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Option 1: GUI (Easiest)
  1. Run: python gui/sparrow_gui.py
  2. Upload policy document
  3. Click "🎯 Analyze Document"
  4. Go to "📚 Appendices" tab
  5. View/Copy/Download appendices

Option 2: Python API (Flexible)
  from appendices_generator import AppendicesGenerator
  
  generator = AppendicesGenerator()
  appendices = generator.generate_all_appendices(
      analysis=my_analysis_dict,
      document_title="Policy Title"
  )
  
  # Save to disk
  files = generator.save_appendices(appendices, '/output/path')

Option 3: Automatic (Pipeline)
  from narrative_integration import NarrativeGenerationPipeline
  
  pipeline = NarrativeGenerationPipeline(generate_appendices=True)
  result = pipeline.generate_complete_narrative(analysis)
  
  # Appendices automatically included
  appendices = result['appendices']

Option 4: Testing
  python test_appendices_generation.py
